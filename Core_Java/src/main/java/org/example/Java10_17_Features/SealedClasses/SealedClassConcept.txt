SEALED CLASSES IS USED TO PERMIT SOME OF THE CLASSES TO EXTEND IT
SEALED CLASSES MUST HAVE TO PERMIT ATLEAST ONE CLASS BEFORE CREATING IT TO SEALED TYPE
THE EXTENDED CLASS CAN BE OF TYPE sealed, non-sealed, final
ONLY THE CLASSES THAT ARE PERMITTED CAN EXTEND THE SEALED CLASS BUT HAVE TO BE EXTEND BY ALL THE BASE CLASS ELSE GIVES ERROR
SEALED CLASS CAN EXTENDS ANOTHER CLASS THEN WILL GIVE PERMITS TO DIFFERENT SUB CLASSES
