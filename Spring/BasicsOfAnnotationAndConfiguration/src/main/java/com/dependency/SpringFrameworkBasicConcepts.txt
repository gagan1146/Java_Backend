->  DEPENDENCY INJECTION :- IS A DESIGN PATTERN THAT IS BASICALLY USED TO INJECT DEPENDENCY OF ONE OBJECT INTO ANOTHER OBJECT...
->  IT CAN BE INJECTED BY THREE WAYS :-
    ->  XML BASED
    ->  JAVA BASED
    ->  ANNOTATION BASED


-> BEAN LIFE CYCLE
    -> BEAN OBJECT CREATED
    -> BEAN OBJECT INSTANTIATION
    ->


-> AUTOWIRING :- IT IS BASICALLY A FEATURE OF SPRING FRAMEWORK THROUGH WHICH DEPENDENCY INJECTION IS PERFORMED AUTOMATICALLY
    ->  AUTO MEANS MAKING A CONNECTION BETWEEN OBJECTS AUTOMATICALLY
    ->  WIRING MEANS LINKING THOSE OBJECTS TO FULFILL DEPENDENCIES
    -> IT CAN BE ACHIEVED BY TWO WAYS
        -> ANNOTATION BASED - @Autowired
        -> XML BASED        - autowired attribute


-> IOC :- INVERSION OF CONTROL
    ->
